# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial release of surfgeo Python SDK
- Core tracking client (sync + async)
- Django middleware integration
- Flask extension
- FastAPI middleware
- WSGI and ASGI generic middleware
- Type hints throughout
- Comprehensive test suite
- Complete documentation
- Example applications for Django, Flask, and FastAPI

### Features

- Non-blocking, fire-and-forget tracking
- Zero impact on application performance
- Automatic request metadata capture
- Configurable timeout (10-100ms)
- Debug mode for development
- Support for Python 3.8+
- Works with Django, Flask, FastAPI, and any WSGI/ASGI framework
